Introduction

In this paper, we introduce a novel Open-Vocabulary GMOT, called Open-GMOT, which can track never-seen object categories with zero training examples. Our Open-GMOT encompasses two main contributions, addressing both the object detection and object association stages. In the first stage, we introduce Open-CSOD (Open-Vocabulary Characteristic-Specific Object Detection), a pioneering method for detecting unseen objects of a generic type. In the second stage, we propose MAC-SORT (Motion-Appearance Cost SORT), an inventive tracking algorithm that seamlessly integrates visual appearance with motion-based matching.
